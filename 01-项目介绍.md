# 第一章：项目介绍

## 1.1 什么是 Claude Code

Claude Code 是 Anthropic 公司开发的官方 CLI 编程助手，它可以在终端中与用户交互，帮助用户完成编程任务。这个项目 (`claude-code-best/claude-code`) 是对 Anthropic 官方 Claude Code CLI 工具的**反编译/逆向还原项目**。

### 项目目标

1. **功能复现**: 尽可能还原 Claude Code 的核心功能
2. **学习研究**: 帮助开发者理解 AI 编程助手的工作原理
3. **社区贡献**: 允许社区参与改进和扩展

### 项目命名

这个项目被戏称为 **CCB (踩踩背)**，这是一种幽默的命名，暗示这个项目是对原始 Claude Code 的"踩踏"（致敬和挑战）。

## 1.5 与官方版本的差异

### 主要差异

1. **feature() 函数**: 所有功能标志默认返回 `false`
   - 原始代码���用 `feature('FLAG_NAME')` 进行条件编译
   - 反编译版本中该函数始终返回 `false`，禁用了所有可选功能

2. **类型错误**: 约 1341 个 TypeScript 编译错误
   - 这些来自反编译过程
   - 不影响 Bun 运行时执行

3. **构建产物**: 约 25MB 的单文件 bundle
   - 使用 Bun 构建
   - 支持 Bun 和 Node.js 运行时

### 设计权衡

| 方面 | 官方版本 | 反编译版本 |
|------|---------|-----------|
| 功能完整性 | 100% | ~70% |
| 类型安全 | 完全 | 部分丢失 |
| 可维护性 | 官方维护 | 社区维护 |
| 更新速度 | 官方更新 | 手动同步 |

## 1.6 快速开始

### 环境要求

```bash
# 必须使用最新版本的 Bun
bun upgrade
```

最低要求：
- Bun >= 1.3.11
- 支持 ANSI 颜色的终端

### 安装步骤

```bash
# 1. 克隆仓库
git clone https://github.com/claude-code-best/claude-code.git
cd claude-code

# 2. 安装依赖
bun install

# 3. 开发模式运行
bun run dev

# 4. 构建生产版本
bun run build
```

### 验证安装

```bash
# 查看版本号 (应该是 2.1.888)
bun run dev --version

# 或者
claude-js --version
```
