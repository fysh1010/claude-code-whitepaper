## 14.2 Buddy 宠物系统详解

### 14.2.1 功能概述

Buddy 是 Claude Code 2.1 版本新增的宠物/伙伴系统，它是一个终端内的 ASCII 动画角色，会在使用 Claude Code 时陪伴用户。这个系统设计精美，包含丰富的自定义选项。

**核心特性**：
- **18 种物种**：鸭子、鹅、水滴、猫、龙、章鱼、猫头鹰、企鹅、乌龟、蜗牛、幽灵、蝾螈、水豚、仙人掌、机器人、兔子、蘑菇、胖达
- **6 种稀有度**：普通（60%）、稀有（25%）、稀有（10%）、史诗（4%）、传奇（1%）
- **6 种眼睛样式**：·、✦、×、◉、@、°
- **8 种帽子**：无、王冠、礼帽、螺旋桨、光环、巫师帽、毛线帽、小鸭子
- **5 种属性**：DEBUGGING（调试）、PATIENCE（耐心）、CHAOS（混沌）、WISDOM（智慧）、SNARK（毒舌）
- **闪光变异**：1% 概率出现闪光的特殊变体

### 14.2.4 伙伴生成算法

**文件**：`src/buddy/companion.ts`

Buddy 使用确定性随机数生成器，基于用户 ID 生成伙伴属性，确保每个用户获得独特但稳定的伙伴：

```typescript
// Mulberry32 — 轻量级随机数生成器
function mulberry32(seed: number): () => number {
  let a = seed >>> 0
  return function () {
    a |= 0
    a = (a + 0x6d2b79f5) | 0
    let t = Math.imul(a ^ (a >>> 15), 1 | a)
    t = (t + Math.imul(t ^ (t >>> 7), 61 | t)) ^ t
    return ((t ^ (t >>> 14)) >>> 0) / 4294967296
  }
}

// 字符串哈希
function hashString(s: string): number {
  if (typeof Bun !== 'undefined') {
    return Number(BigInt(Bun.hash(s)) & 0xffffffffn)
  }
  let h = 2166136261
  for (let i = 0; i < s.length; i++) {
    h ^= s.charCodeAt(i)
    h = Math.imul(h, 16777619)
  }
  return h >>> 0
}

// 根据用户 ID 生成伙伴属性
export function roll(userId: string): Roll {
  const key = userId + 'friend-2026-401'
  const value = rollFrom(mulberry32(hashString(key)))
  return value
}

// 获取用户的伙伴
export function getCompanion(): Companion | undefined {
  const stored = getGlobalConfig().companion
  if (!stored) return undefined
  const { bones } = roll(companionUserId())
  return { ...stored, ...bones }
}
```

**渲染函数**：

```typescript
export function renderSprite(bones: CompanionBones, frame = 0): string[] {
  const frames = BODIES[bones.species]
  const body = frames[frame % frames.length]!.map(line =>
    line.replaceAll('{E}', bones.eye),  // 替换眼睛占位符
  )
  const lines = [...body]
  // 帽子处理
  if (bones.hat !== 'none' && !lines[0]!.trim()) {
    lines[0] = HAT_LINES[bones.hat]
  }
  return lines
}

export function renderFace(bones: CompanionBones): string {
  const eye: Eye = bones.eye
  switch (bones.species) {
    case cat:
      return `=${eye}ω${eye}=`
    case dragon:
      return `<${eye}~${eye}>`
    // ... 更多物种的脸部渲染
  }
}
```

### 14.2.7 如何启用 Buddy

在当前反编译版本中，`feature('BUDDY')` 始终返回 `false`。有两种方法可以启用：

**方法一：修改 feature() 函数**

编辑 `src/entrypoints/cli.tsx`：

```typescript
function feature(flag: string): boolean {
  const enabledFeatures: Record<string, boolean> = {
    'BUDDY': true,  // 启用 Buddy 系统
  }
  return enabledFeatures[flag] ?? false
}
```

**方法二：修改配置文件**

在 `~/.claude/settings.json` 中添加伙伴数据：

```json
{
  "companion": {
    "name": "Buddy",
    "personality": "Helpful and curious, always ready to assist",
    "hatchedAt": 1704067200000
  },
  "companionMuted": false
}
```

export const memoryCommand = {
  name: 'memory',

### 14.4.1 功能概述

Bridge 是 Claude Code 的远程控制功能，允许用户通过 Web 界面或其他客户端远程控制 Claude Code 会话。这是一个约 12000+ 行的大型模块。

### 14.4.2 源代码实现

**文件**：`src/bridge/initReplBridge.ts`

```typescript
// Bridge 初始化和 WebSocket 处理
export function initReplBridge(
  options: BridgeOptions
): ReplBridge {
  return {
    // 建立 WebSocket 连接
    async connect(url: string) {
      const ws = new WebSocket(url)
      ws.on('message', handleRemoteMessage)
      ws.on('close', handleDisconnect)
    },

    // 处理远程命令
    handleRemoteMessage(data: string) {
      const command = JSON.parse(data)
      switch (command.type) {
        case 'input':
          executeInput(command.text)
        case 'interrupt':
          interruptCurrentTask()
        // ...
      }
    },

    // 同步会话状态
    syncState() {
      broadcastState({
        messages: getMessages(),
        cwd: getCwd(),
        toolUsage: getToolUsageStats(),
      })
    }
  }
}
```

### 14.4.3 功能特性

- WebSocket 实时通信
- 远程输入执行
- 状态同步
- 终端输出流式传输
