## 如何阅读本白皮书

### 新手小白建议阅读顺序

1. **先阅读 [02-架构概览](./02-架构概览.md)** - 了解项目整体结构
2. **再阅读 [03-入口与启动](./03-入口与启动.md)** - 理解程序如何运行
3. **然后阅读 [09-使用指南](./09-使用指南.md)** - 学会使用 Claude Code
4. **最后阅读 [05-工具系统](./05-工具系统.md)** - 理解工具如何工作

### 技术人员建议阅读顺序

1. **[02-架构概览](./02-架构概览.md)** - 快速了解结构
2. **[03-入口与启动](./03-入口与启动.md)** - 理解启动机制
3. **[04-查询引擎](./04-查询引擎.md)** - 深入核心逻辑
4. **[05-工具系统](./05-工具系统.md)** - 理解扩展机制
5. **[10-开发指南](./10-开发指南.md)** - 开始开发
6. **[14-隐藏功能与宠物系统](./14-隐藏功能与宠物系统.md)** - 探索隐藏功能
